Overview

This project contains js files that i used to develop my differents projects. Feel free to get theses files as cdn link in your dev project. Please don't use it as a real cdn production. Permanent availability can not be garantuee.

Usage

The URL of js files is http://gauthiermontavon.github.io/js-cdn/js/[js_file.js]

Here is the list of available js files :

  • jquery.trace.js
  • var_dump.js
